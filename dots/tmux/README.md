[![GitHub Repo stars](https://img.shields.io/github/stars/tmux/tmux?style=social)](https://github.com/tmux/tmux)

- Installed by **Homebrew**.
- XDG Base Directory Specification.
- **TPM** as package manager.
- Theme is **Nord**.
